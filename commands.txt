git

  log
    shows log of git updates
    
  status
    shows status of repository
    
  add .
    adds files to staging area (. adds all files, can also only do specific file)
    
  commit -m 'message/description of changes made'
    record changes
    
  push 
    send changes to remote server (GitHub) after committing 
    
  remote 
    add origin <repository_link>
      connect to an existing repository
    -v
      verbose; checks connection with remote git repository
    remove origin
      remove connection to github repo
    set-url origin <git://new.url.here>
      directly set new github connection instead of removing one and adding again
  
  pull
    pulls info from existing repository
  
  
  

other
  
    new file + extension
    
  ls
    list of files in directory
    
  pwd
    see working directory
    
  cd "directory"
    change working directory
  
  mv oldname.ext newname.ext
    change document name
    
  mv "dir/dir/olddir" "dir/dir/newdir"
    change folder name/directory name
